## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| **连接管理** | 增/删/改 SSH 连接，支持密码和密钥认证 |
| **一键测试** | 测试连接是否可达，测试通过后名称变绿 |
| **脚本管理** | 管理远程执行脚本，支持 sh/bash/python/perl 等 |
| **执行中心** | 选择脚本执行 或 直接输入命令执行 |
| **执行历史** | 记录所有执行结果，支持查看详情和清理 |

## 🚀 快速入门

### 1. 新建连接

点击 **＋ 新建** → 填写连接信息：

| 字段 | 说明 |
|------|------|
| 连接名称 | 任意标识名称 |
| 主机地址 | IP 或域名 |
| 端口 | 默认 22 |
| 用户名 | 登录用户名 |
| 认证方式 | 密码或密钥 |
| 密码/密钥路径 | 输入密码，或选择私钥文件 |

### 2. 测试连接

选中连接 → 点击 **🔌 测试**。  
测试成功后，该连接的名称会**变为绿色**，一目了然。

### 3. 管理脚本

点击 **＋ 新建** 创建脚本：

- 选择解释器（sh、bash、python3、perl 等）
- 编写脚本内容
- 保存后在执行中心使用

### 4. 执行命令

两种方式打开执行中心：

- 点击底部 **▶ 执行中心** 按钮
- 在连接列表选中连接 → 点击工具栏 **▶ 执行**

执行中心支持两种模式：

| 模式 | 说明 |
|------|------|
| **执行脚本** | 从已有脚本列表中选择执行 |
| **输入命令** | 直接输入命令行，灵活执行 |

### 5. 查看历史

点击 **📋 执行历史** 查看所有执行记录，支持查看详细输出和删除记录。

## ⚙️ 数据存储

- 数据库文件 `ssh_manager.db` 存放在 exe 同级目录
- 连接信息、脚本、执行历史均存储在此文件中
- 删除 exe + db 即可彻底卸载

---

## 🔒 安全提示

- 连接密码存储在本地 SQLite 数据库中，**非加密存储**
- 请不要在多用户共用的电脑上长期保存敏感密码
- 建议使用密钥认证，安全性更高

## 🐛 常见问题

**Q: 运行时提示"不是有效的 Win32 应用程序"？**  
A: 请确认下载的是 Releases 中的 exe，并且操作系统为 64 位 Windows。

**Q: 测试连接失败？**  
A: 检查用户名密码是否正确，以及目标服务器是否允许密码登录。  
   SSH 服务端需确认 `/etc/ssh/sshd_config` 中 `PasswordAuthentication yes`。

**Q: 密钥认证失败？**  
A: 确认私钥文件格式正确（OpenSSH 格式），如需密码短语请在密钥路径同时填写密码字段。

**Q: 执行命令没有输出？**  
A: 某些命令可能需要交互式终端。尝试在命令前加 `echo "start" &&` 测试。
