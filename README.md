## ✨ 功能一览

| 功能 | 说明 | 入口 |
|------|------|------|
| **连接管理** | 增/删/改 SSH 连接，密码和密钥认证 | 工具栏 + 右键 |
| **🔌 一键测试** | 测试连接可达性，通过后名称变绿 | 工具栏 |
| **脚本管理** | 管理远程执行脚本（sh/bash/python/perl...） | 右侧面板 |
| **⚡ 快捷命令** | 选中连接 → 敲命令 → 回车，一气呵成 | 底部命令栏 |
| **⚡ 快速命令弹窗** | 双击连接弹出精简命令窗口，支持多行 | 双击表格行 / `Ctrl+D` |
| **⚡ 快速执行** | 选中连接一键选脚本执行 | 工具栏 "⚡ 快速执行" |
| **▶ 执行中心** | 完整脚本执行 + 自定义命令 | 工具栏 / `Ctrl+E` |
| **📂 文件浏览** | 远程浏览服务器目录结构 | 工具栏 / 右键 |
| **🔍 搜索过滤** | 按名称或主机地址快速筛选连接 | 右上角搜索框 |
| **📋 执行历史** | 记录所有执行结果 | 工具栏 / `Ctrl+H` |

## 🚀 快速入门

### 1. 新建连接

点击 **＋ 新建** → 填写连接信息：

| 字段 | 说明 |
|------|------|
| 连接名称 | 任意标识名称 |
| 主机地址 | IP 或域名 |
| 端口 | 默认 22 |
| 用户名 | 登录用户名 |
| 认证方式 | 密码或密钥 |
| 密码/密钥路径 | 输入密码，或选择私钥文件 |

### 2. 测试连接

选中连接 → 点击 **🔌 测试** → 名称**变绿**即成功。

### 3. 最快捷操作

```text
选中连接 → 底部输入命令 → 回车       ← 快捷命令
双击连接 → 输入命令 → 执行            ← 快速命令弹窗
选中连接 → 点 ⚡ 快速执行 → 选脚本    ← 快速执行脚本
```

### 4. 文件浏览

选中连接 → 点 **📂 文件** → 双击目录进入 → ⬆ 返回上级

### 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+D` | 快速命令 |
| `Ctrl+E` | 执行中心 |
| `Ctrl+H` | 执行历史 |
| `Ctrl+Q` | 退出 |

## ⚙️ 数据存储

- 数据库文件 `ssh_manager.db` 存放在 exe 同级目录
- 连接信息、脚本、执行历史均存储在此文件中
- 删除 exe + db 即可彻底卸载

---

## 🔒 安全提示

- 连接密码存储在本地 SQLite 数据库中，**非加密存储**
- 请不要在多用户共用的电脑上长期保存敏感密码
- 建议使用密钥认证，安全性更高

## 🐛 常见问题

**Q: "不是有效的 Win32 应用程序"？**  
A: 请确认下载的是 64 位版 exe。

**Q: 测试连接失败？**  
A: 检查用户名密码，确保服务端 `PasswordAuthentication yes`。

**Q: 密钥认证失败？**  
A: 确认私钥为 OpenSSH 格式，密码短语请在密码字段填写。
